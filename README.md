## 📖 About

CarbonIT Global is a platform dedicated to sustainable carbon solutions. This repository contains the fully redesigned frontend of the official website — built with **React**, featuring user authentication, a personal dashboard, carbon calculation tools, blog content, and more.

---

## ✨ Features

- 🏠 **Home Page** — Hero section, highlights, and call-to-action
- 🔐 **Authentication System** — Signup, Login, Forgot Password, New Password, and Email Verification flows
- 📊 **User Dashboard** — Personalized user area after login
- 🧮 **Carbon Calculator (Hesaplama)** — Calculate carbon footprint interactively
- 📰 **Blog** — Articles and updates on carbon sustainability
- 💡 **Solutions** — Overview of services and offerings
- 🎬 **Demo** — Product/service demonstration section
- 📦 **SKDM** — Dedicated section for SKDM-related content
- 📞 **Contact** — Contact form and company information
- 🧭 **Navbar & Footer** — Consistent navigation and footer across all pages

## ⚙️ Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or above recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

```bash
# Clone the repository
git clone https://github.com/Bilgisim-Web-Sites/CarbonIT-website.git

# Navigate into the project
cd CarbonIT-website/Work

# Install dependencies
npm install

# Start the development server
npm start
```

The app will be available at `http://localhost:3000`.
