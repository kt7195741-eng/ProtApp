GreenTrack

GreenTrack is a full-stack web application designed to help users actively manage their personal plant collections while measuring their positive impact on the environment. By logging and maintaining different plant species, users can track their contribution to reducing carbon emissions and compete on a community leaderboard based on the CO2 they have saved.

Key Features:

    Secure User Management: Features JWT-based authentication, password recovery via email verification, and secure profile management (including safe "soft deletes" for accounts).

    Plant Dashboard: A visual interface where users can add plants, upload custom photos, specify plant types and species, and set watering frequencies.

    Eco-Impact Tracking: Calculates and displays the estimated kilograms of CO2 saved by each user's plant collection.

    Community Leaderboard: A gamified ranking system that highlights the top eco-friendly users based on their environmental contributions.

Tech Stack:

    Frontend: React.js

    Backend: Java Spring Boot, Spring Security (JWT)

    Database: PostgreSQL
